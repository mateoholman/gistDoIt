Gist Do It

Objective

To demonstrate the combined knowledge gained in the ACA Intermediate Coursework

Skills Learned

  • AJAX calls
  • Retrieving and manipulating JSON data
  • GitHub API
  • Bootstrap Framework

Libraries & Packages Used

  • marked.js
  • Bootstrap
  • http server

Process

  1. Use jQuery to make an AJAX call and review the GitHub JSON object.
  2. Extract the description from each blog tagged as a post and create a list of the posts in a sidebar.
  3. Use jQuery to create a click event listener that will ignore the default action when an anchor is clicked and instead, update the #post div section of our blog with the post content within our JSON object.
  4. Use the marked.js library to convert the post content to html
  5. Make another AJAX call to the JSON object, fetch the comment data and append it within a comment section.
  6. Style the blog with Bootstrap and some custom CSS.
